Como utilizar os Hiperlinks ou Links

O que é um Hiperlink/Link?

Hiperlinks, de forma resumida, são as referências dentro de um documento para outras partes do mesmo documento, outros documentos, arquivos ou endereço eletrônico.

Podem ser remotos, internos, locais, com endereço de correio ou arquivo.

Marcação

A tag para marcação de hiperlinks é: < a > propriedade href que faz a referência para ao conteúdo desejado.

Assim, a sintaxe completa fica: < a href="endereço" > Conteúdo Referenciado Clicável < /a >

Hiperlinks Externos ou Remotos

Para utilizar os Links que referenciam outros sites ou páginas da web, basta adicionar na propriedade href utilizamos a URL para a referência que deseja.

Exemplo: IFPR

Para isso fazemos: < a href="https://irati.ifpr.edu.br/?msclkid=4b754342d08611ecbc70a38a26356d94" > IFPR < /a >

Hiperlinks Internos

Os Hiperlinks Internos são links que apontam a um lugar diferente dentro da mesma página. Para isso, podermos fazer de duas formas:

  1. (no local Clicável) < a href="#referencia" > Conteúdo Referenciado < /a >

    2. (no texto a ser direcionado) < a name="referencia" > < /a >

  2. < a href="#referencia" > Conteúdo Referenciado < /a >

    3. tag id="referencia " >

Exemplo: Hiperlinks Internos

Hiperlinks Locais

São links que apontam para outros páginas dentro de um mesmo site. Para isso usamos na sintaxe:

< a href=“arquivo.html” > Conteúdo Referenciado < /a >

Hiperlink em Imagem

Envolvendo a tag de imagem com a tag de hiperlink é possível criar hiperlinks imagens.

< a href=“referencia” > < img src="imagem.jpg"> < /a >

Hyperlinks para endereço eletrônico

São aqueles que apontam para um endereço eletrônico

Exemplo:

gmail@gmail.com

Para isso a sintaxe é:

< a href= "mailto: gmail@gmail.com" > gmail@gmail.com < /a >
